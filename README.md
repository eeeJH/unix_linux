# unix_linux

`unix` 와 `linux` 의 차이점 



#### 명령어

- `cd`  directory  ==  change directory

- `mkdir`  directory  ==  make directory

- `ls`  ==  show list
- `touch`  file  ==  create a file
- `mv`  file1 file2  ==  move file1 to file2 (directory 도 가능)
- `rm`  file  ==  remove a file
- `rm -rf`  directory  ==  remove a directory
